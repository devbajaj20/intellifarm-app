# 🌱 IntelliFarm – Smart Agriculture System

🚀 *AI-powered platform for intelligent farming and data-driven agricultural decisions*

---

## 📌 Overview

**IntelliFarm** is an intelligent agricultural assistance system that leverages **Machine Learning** and **Deep Learning** to help farmers make smarter decisions. It integrates multiple AI modules into a single platform to improve productivity, optimize resources, and promote sustainable farming.

---

## ✨ Features

* 🌾 **Crop Recommendation**
  Suggests the best crops based on soil nutrients and environmental conditions

* 🧪 **Fertilizer Recommendation**
  Provides optimal fertilizer suggestions using machine learning models

* 🌿 **Plant Disease Detection**
  Detects plant diseases using deep learning (CNN) from leaf images

* 🤖 **AI Chatbot Assistant**
  Helps farmers with real-time agricultural advice

* 📱 **Mobile Application**
  User-friendly interface built using Flutter

---

## 🧠 Technologies Used

* **Frontend:** Flutter
* **Backend:** Python (Flask / FastAPI)
* **Machine Learning:** Scikit-learn
* **Deep Learning:** TensorFlow / Keras
* **Database & Cloud:** Firebase

---

## 🏗️ System Architecture

The system follows a multi-layer architecture:

1. **Data Collection** – Soil, weather, and crop images
2. **Data Processing** – Cleaning, normalization, feature extraction
3. **AI Models** – ML for prediction, CNN for image classification
4. **Knowledge Layer** – Chatbot and advisory system
5. **Application Layer** – Mobile app interface

---

## 📊 Model Performance

* ✅ Crop Recommendation Accuracy: **97.5%** 
* ✅ Fertilizer Recommendation Accuracy: **96.8%** 
* ✅ Disease Detection Accuracy: **96%+**

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/devbajaj20/intellifarm.git
cd intellifarm
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run backend

```bash
python app.py
```

### 4️⃣ Run mobile app

```bash
flutter run
```
